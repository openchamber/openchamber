⚠️ OpenCode + Anthropic auth — những gì bạn cần biết ngay Tình hình: OpenCode v1.3.0 sẽ xóa Anthropic auth (Anthropic gửi luật sư). v1.2.27 hiện tại vẫn OK. Giải pháp ngay: 1. Tắt autoupdate + thêm plugin vào ~/.config/opencode/opencode.json: 
{
  "autoupdate": false,
  "plugin": ["opencode-claude-auth"]
}
2. Muốn tự maintain lâu dài thì fork: 
git clone https://github.com/sst/opencode
cd opencode && git checkout 8e09e8c61
git checkout -b maintain/with-anthropic-auth
# push lên repo cá nhân → set private
TL;DR: Dùng v1.2.27 + opencode-claude-auth + tắt autoupdate là đủ. Fork chỉ cần nếu muốn tự kiểm soát hoàn toàn.
Không có drama. Chỉ cần action trước khi lỡ update. 🔒
