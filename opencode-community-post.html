WHAT HAPPENED

! Anthropic đã gửi luật sư tới SST. OpenCode v1.3.0 sẽ xóa Anthropic auth.
  • Version v1.3.0+ — Anthropic auth bị remove hoàn toàn
  • Version v1.2.27 — hiện tại vẫn hoạt động bình thường
  • Autoupdate = rủi ro, có thể update lên v1.3.0 bất cứ lúc nào
// giải pháp 1 · dễ nhất

CONFIG + TẮT AUTOUPDATE

Thêm opencode-claude-auth vào plugin list và tắt autoupdate trong file config. Đây là cách nhanh nhất — không cần fork hay rebuild.

json · opencode config 
{
  "autoupdate": false,
  "plugin": ["opencode-claude-auth"]
}
  • Tắt autoupdate trước — ưu tiên số 1
  • Plugin opencode-claude-auth giữ auth flow hoạt động
  • Không cần rebuild hay compile gì thêm
// giải pháp 2 · cho ai muốn control cao hơn

FORK TẠI COMMIT ỔN ĐỊNH

Clone repo, checkout đúng commit 8e09e8c61 trước khi auth bị xóa, tạo branch riêng và push lên repo cá nhân private.

bash · fork & preserve 
git clone https://github.com/sst/opencode
cd opencode && git checkout 8e09e8c61
git checkout -b maintain/with-anthropic-auth
# push lên repo cá nhân → set private
  • Commit 8e09e8c61 — điểm cuối trước khi auth bị strip
  • Set repo private để tránh vi phạm ToS
  • Phù hợp nếu bạn cần customize thêm hoặc tự build
TL;DR · action plan

Dùng v1.2.27 + opencode-claude-auth plugin + tắt autoupdate là đủ cho 99% trường hợp. Fork chỉ cần thiết nếu muốn self-host hoặc maintain long-term.
