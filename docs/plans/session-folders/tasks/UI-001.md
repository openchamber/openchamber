## Description

Tạo React component render một folder header cùng với danh sách sessions bên trong. Component nhận callback `renderSessionNode` từ `SessionSidebar` để render sessions — tái sử dụng toàn bộ logic hiện có, không duplicate.

---

## Props Interface

```
Props:
  - folder: SessionFolder         (từ useSessionFoldersStore types)
  - sessions: SessionNode[]       (đã được filter theo folder.sessionIds)
  - isCollapsed: boolean
  - onToggle: () => void          (toggle collapse)
  - onRename: (name: string) => void
  - onDelete: () => void
  - renderSessionNode: (node: SessionNode, depth?: number, groupDir?: string | null, projectId?: string | null) => React.ReactNode
  - groupDirectory?: string | null  (truyền xuống renderSessionNode)
  - projectId?: string | null
  - mobileVariant?: boolean
```

## UI Structure

### Folder Header

Layout: `[folder-icon] [name hoặc rename-input] [count badge] [collapse-arrow] [action-buttons]`

- **Folder icon**: `RiFolderLine` (collapsed) / `RiFolderOpenLine` (expanded) — từ `@remixicon/react`
- **Name**: text, click vào mở inline rename
- **Count badge**: số sessions trong folder, dạng `(3)`, dùng `typography-micro text-muted-foreground`
- **Collapse arrow**: `RiArrowRightSLine` / `RiArrowDownSLine`
- **Action buttons** (hiện khi hover, pattern giống group header hiện tại):
  - Rename button (pencil icon `RiPencilAiLine`)
  - Delete button (bin icon `RiDeleteBinLine`, text-destructive khi hover)

### Inline Rename

Khi đang rename, thay thế tên bằng `<input>`:
- `autoFocus`
- Submit bằng Enter
- Cancel bằng Escape
- Dùng `data-keyboard-avoid="true"` (pattern hiện có trong SessionSidebar)

### Folder Body

Khi expanded: render `sessions.map(node => renderSessionNode(node, 0, groupDirectory, projectId))`

Empty state khi `sessions.length === 0`:
- Text: `"Empty folder"` — `typography-micro text-muted-foreground/70`
