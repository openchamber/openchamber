## Cấu trúc submenu cần thêm

```
DropdownMenuSeparator   ← ngăn cách với các items trên

DropdownMenuSub

    DropdownMenuSeparator  ← nếu có folders
    
    DropdownMenuItem: "New folder..."
      icon: RiAddLine
      onClick: mở inline new folder input (state sẽ handle ở SS-004)

        icon: RiFolderMinusLine (hoặc RiFolderLine)
```

## Data cần lấy bên trong renderSessionNode

Callback `renderSessionNode` cần access thêm:
- `activeProjectId` — để biết projectId (đã có trong scope của SessionSidebar)
- `getFoldersForProject(activeProjectId)` → danh sách folders của project
- `getSessionFolderId(activeProjectId, session.id)` → folder hiện tại của session (nếu có)
- `addSessionToFolder`, `removeSessionFromFolder`, `createFolder` actions

> Vì `renderSessionNode` là `React.useCallback` với dependency array, cần thêm các dependencies mới vào mảng deps của `useCallback`.
