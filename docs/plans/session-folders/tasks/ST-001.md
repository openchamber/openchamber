## Description

Tạo Zustand store quản lý toàn bộ logic cho Session Folders. Store này lưu folders theo từng project (scoped by projectId) bằng `getSafeStorage()` — pattern giống `useProjectsStore.ts`.

---

## Types cần định nghĩa

```
Interface SessionFolder:
  - id: string            (dùng crypto.randomUUID())
  - name: string          (tên folder, do user đặt)
  - sessionIds: string[]  (session IDs thuộc folder này)
  - createdAt: number     (timestamp)

Type SessionFoldersMap = Record<projectId: string, SessionFolder[]>
```

---

## State Shape

```
State:
  - foldersMap: SessionFoldersMap    // all folders, keyed by projectId
  - collapsedFolderIds: Set<string>  // folder IDs đang collapsed (global across projects)
```

---

## Actions cần implement

### `getFoldersForProject(projectId: string): SessionFolder[]`
- Selector: trả về `foldersMap[projectId] ?? []`

### `createFolder(projectId: string, name: string): SessionFolder`
- Tạo folder mới với unique ID
- Append vào `foldersMap[projectId]`
- Persist vào storage
- Return folder mới

### `renameFolder(projectId: string, folderId: string, name: string): void`
- Update tên folder trong `foldersMap[projectId]`
- Persist

### `deleteFolder(projectId: string, folderId: string): void`
- Xóa folder khỏi `foldersMap[projectId]`
- **Không xóa sessions** — sessions trở thành ungrouped
- Persist

### `addSessionToFolder(projectId: string, folderId: string, sessionId: string): void`
- **Auto-remove** session khỏi folder cũ nếu có (1 session chỉ thuộc 1 folder)
- Add sessionId vào folder target
- Persist

### `removeSessionFromFolder(projectId: string, sessionId: string): void`
- Xóa sessionId khỏi bất kỳ folder nào nó đang thuộc trong project
- Persist

### `toggleFolderCollapse(folderId: string): void`
- Toggle `collapsedFolderIds` set
- Persist collapsed state vào storage (key riêng)

### `cleanupSessions(projectId: string, existingSessionIds: Set<string>): void`
- Với mỗi folder trong project: filter `sessionIds` để chỉ giữ IDs còn trong `existingSessionIds`
- Persist nếu có thay đổi
- Dùng để cleanup khi sessions bị xóa

### `getSessionFolderId(projectId: string, sessionId: string): string | null`
- Tìm folder nào chứa session này trong project
- Return folderId hoặc null nếu ungrouped
