## Description

## Logic thay đổi trong `renderGroupSessions`

Hàm `renderGroupSessions` hiện tại (line ~2164) render tất cả sessions của một group. Cần sửa để:

### Bước 1: Phân loại sessions

Trong `renderGroupSessions`, trước khi render:

```
foldersForProject = getFoldersForProject(activeProjectId)

// Tập hợp sessionIds đã có folder
sessionIdsInFolders = new Set(
  foldersForProject.flatMap(folder => folder.sessionIds)
)

// Sessions không thuộc folder nào (ungrouped)
ungroupedSessions = group.sessions.filter(
  node => !sessionIdsInFolders.has(node.session.id)
)
```

### Bước 2: Với mỗi folder, filter sessions thuộc folder

```
folderSessionNodes = folder.sessionIds
  .map(sessionId => group.sessions.find(node => node.session.id === sessionId))
  .filter(Boolean)
```

### Bước 3: Render structure mới

Thay vì render tất cả `group.sessions`:

```
render:
  [danh sách SessionFolderItem — với mỗi folder có session trong group này]
  [ungroupedSessions như cũ]  ← dùng lại `renderSessionNode`
  [Show X more button — áp dụng cho ungroupedSessions nếu cần]
```

### Scope quan trọng

Một folder được hiển thị **chỉ trong group** nếu folder đó có ít nhất 1 session trong group hiện tại. Điều này tự nhiên vì sessions đã được scoped by directory trong group.

## Truyền props vào SessionFolderItem

```
<SessionFolderItem
  key={folder.id}
  folder={folder}
  sessions={folderSessionNodes}
  isCollapsed={collapsedFolderIds.has(folder.id)}
  onToggle={() => toggleFolderCollapse(folder.id)}
  onRename={(name) => {
    if (activeProjectId) renameFolder(activeProjectId, folder.id, name)
  }}
  onDelete={() => {
    if (activeProjectId) deleteFolder(activeProjectId, folder.id)
  }}
  renderSessionNode={renderSessionNode}
  groupDirectory={group.directory}
  projectId={projectId}
  mobileVariant={mobileVariant}
/>
```

## Edge cases

- **Folder rỗng trong group này**: Không hiển thị folder đó trong group này (vì không có session nào)
- **Session trong folder nhưng folder bị xóa**: Không xảy ra nếu `deleteFolder` cleanup đúng (store xóa folder, sessions tự ungrouped)
- **"Show X more" button**: Chỉ apply cho `ungroupedSessions`, không tính sessions trong folders
