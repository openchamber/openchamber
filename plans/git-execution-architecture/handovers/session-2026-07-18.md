- Current used 21.311% fewer top-level Git launches than historical but completed 38.588–41.621% more slowly under the broad bounded workload on this machine.
- Current+fsmonitor cold/warm p95 was 0.504–2.452% lower than current without fsmonitor, while whole-workload duration was 4.839–6.378% longer and Git launches were unchanged.
- Both orders recorded 300 cold, 300 warm, and 1,260 conservative mutation-refresh hook responses; all 200 repository configs remained unchanged.
