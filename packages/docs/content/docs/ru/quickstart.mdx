# Быстрый старт

## Самый быстрый путь

1. Установите [OpenCode](https://opencode.ai).
2. Установите OpenChamber CLI (см. [Установка](/install/) для однострочной команды).
3. Запустите `openchamber --ui-password be-creative-here`.
4. Откройте URL, который выведет CLI (обычно `http://localhost:3000`).
5. Чтобы пользоваться с телефона, запустите [туннель](/tunnels/) и отсканируйте QR-код.

В браузере должен появиться список сессий OpenChamber. Если страница загрузилась — всё готово.

Используйте надёжный пароль UI, особенно если планируете открыть инстанс в интернет.

Если страница не загружается, см. [Устранение неполадок](/troubleshooting/).

## Какое приложение выбрать?

- **desktop** — для повседневной работы на macOS
- **web** — для удалённого доступа и работы с телефона
- **VS Code** — для сессий рядом с вашим кодом
