# Действия проекта

Действие проекта — это команда оболочки, которую вы сохраняете один раз и запускаете кликом — ваш dev-сервер, сборка, прогон тестов. Каждый проект хранит собственный список. Настройте их в **Настройки → Проекты → Действия проекта**.

## Добавление действия

1. Откройте **Настройки → Проекты** и найдите раздел **Project Actions**.
2. Добавьте действие, задайте имя, выберите значок и введите команду для запуска.
3. Сохраните.

Можно ограничить действие определёнными операционными системами, если команда имеет смысл только на одной.

## Запуск действия

Действия располагаются в меню в шапке приложения. Нажмите одно, и OpenChamber запустит его в терминале в каталоге проекта, переключая вас на представление терминала, чтобы видеть вывод. Остановите его из того же меню.

## Автооткрытие dev-сервера

Включите **auto-open URL** для действия, которое запускает сервер. OpenChamber следит за выводом в поисках локального адреса и предлагает открыть его — см. [Предпросмотр и dev-серверы](/preview/). В десктопе можно также направить его через SSH-порт-forward.
