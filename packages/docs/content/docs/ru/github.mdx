# Issues и PR GitHub

Подключите свой аккаунт GitHub, и OpenChamber сможет подтягивать issues и pull-реквесты, начинать сессию прямо из них, а также открывать или обновлять PR для вас.

## Подключение GitHub

1. Откройте **Настройки → Git**.
2. В разделе GitHub выберите **Connect**. OpenChamber покажет ссылку и короткий код.
3. Откройте ссылку, введите код и подтвердите.

Когда подключение выполнено, ваш аккаунт отображается в разделе GitHub. Вы можете подключить несколько аккаунтов и переключаться между ними или отключить в любой момент.

## Начало работы из issue или PR

Когда вы создаёте [worktree-сессию](/worktrees/) с подключённым GitHub, можно выбрать **Start from GitHub issue/PR**:

- выберите **issue**, и OpenChamber назовёт ветку по нему и откроет сессию с issue и его комментариями первым сообщением
- выберите **pull request**, и он переключится на ветку PR; можно включить diff PR, чтобы у агента было полное изменение

Это переносит вас прямо в сессию с уже загруженным контекстом.

## Открытие и управление pull-реквестами

С вкладки **PR** в [git-представлении](/git/) можно создать pull-реквест, обновить его, пометить драфт как готовый или слить. OpenChamber может сгенерировать заголовок и описание PR из ваших изменений.
