# MCP-серверы

MCP-сервер даёт вашим агентам дополнительные инструменты — например, поиск по базе данных, вызов API или чтение сервиса, которым вы пользуетесь. Добавляйте их в **Настройки → MCP**.

## Добавление сервера

1. Откройте **Настройки → MCP**.
2. Добавьте сервер и выберите его тип:
   - **local** — OpenChamber запускает команду на вашей машине. Вы даёте команду для запуска и, при необходимости, переменные окружения.
   - **remote** — OpenChamber подключается к URL, который хостит кто-то другой. Вы даёте URL и нужные заголовки (например, auth-токен).
3. Сохраните. Сервер включён по умолчанию; можно выключить его без удаления.

## Где применяется

Выберите область при добавлении сервера:

- **personal** — доступен в каждом проекте
- **project** — доступен только в текущем проекте и сохраняется вместе с остальными настройками проекта

Имена серверов используют строчные буквы, цифры, дефисы и подчёркивания.
