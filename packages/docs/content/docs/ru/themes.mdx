# Темы

OpenChamber поддерживает встроенные темы и пользовательские JSON-файлы тем.

## Добавить пользовательскую тему

1. Создайте каталог тем:

```bash
mkdir -p ~/.config/openchamber/themes
```

2. Добавьте ваш JSON-файл в этот каталог (например, `my-theme.json`).
3. Откройте OpenChamber, затем перейдите в **Settings -> Theme -> Reload themes**.
4. Выберите вашу тему из выпадающего списка. Она применится сразу.

## Расположение тем

- macOS/Linux: `~/.config/openchamber/themes/`
