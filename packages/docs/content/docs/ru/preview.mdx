# Предпросмотр и dev-серверы

Когда вы запускаете dev-сервер, OpenChamber может открыть его прямо внутри приложения, а не в отдельной вкладке браузера — чтобы видеть сайт рядом с чатом, захватывать его консоль и указывать на элементы, чтобы спрашивать о них.

## Открытие предпросмотра

OpenChamber следит за выводом терминала в поисках локального адреса (строка `Local:`, которую печатают инструменты вроде Vite, Next.js или Astro). Обнаружив его:

- в терминале появляется кнопка **Open preview**
- [действие проекта](/project-actions/) с включённым автооткрытием открывает его за вас
- локальная ссылка в сообщении чата тоже может открыть его

Сайт загружается в боковой панели. Предпросмотр возможен только для локальных адресов (на вашей машине).

## Консоль и инспектор

В панели предпросмотра вы можете:

- следить за **консолью** страницы — ошибки, предупреждения и логи, с любой фильтрацией
- включить **inspect**, кликнуть любой элемент и отправить заметку о нём — селектор, стили, позицию и скриншот — прямо в чат

Это самый быстрый способ сказать агенту «вот эта кнопка, здесь» без описания словами.
