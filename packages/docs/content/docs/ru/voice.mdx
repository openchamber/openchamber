# Голосовой режим

Голосовой режим позволяет диктовать сообщения и прослушивать ответы. Включается в **Settings → OpenChamber → Voice**.

## Чтение ответов вслух (text-to-speech)

Выберите, как озвучиваются ответы:

- **browser** — встроенные голоса браузера, без настройки
- **OpenAI** — голоса OpenAI; вставьте ваш API-ключ и выберите голос
- **OpenAI-compatible** — любой сервис, совместимый с форматом OpenAI; введите его URL и при необходимости API-ключ
- **macOS say** — встроенная команда `say`, на Mac, где она есть

После включения на сообщениях появляется кнопка воспроизведения, чтобы прослушать их вслух.

## Диктовка сообщений (speech-to-text)

Выберите, как расшифровывается ваша речь:

- **browser** — встроенное распознавание браузера, без настройки
- **server** — OpenAI-совместимый сервис транскрипции; введите его URL и при необходимости API-ключ
- **on-device** — речевая модель, работающая в браузере; скачивается при первом использовании

## Заметка для телефонов

Для чтения ответов вслух на телефоне наиболее надёжны варианты OpenAI или OpenAI-compatible — мобильные браузеры ограничивают встроенные голоса.
