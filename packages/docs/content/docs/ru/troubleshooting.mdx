# Устранение неполадок

Возникла проблема? Найдите симптом ниже и попробуйте исправление.

## Команда OpenChamber завершается или не запускается

- подтвердите Node.js `>=22`
- выполните `openchamber --version`
- при необходимости переустановите последний CLI

## Веб-UI недоступно

- проверьте логи сервера командой `openchamber logs`
- проверьте активный порт (по умолчанию `3000`)
- сначала откройте `http://localhost:3000` напрямую, прежде чем тестировать туннельные ссылки

## Удалённая/туннельная ссылка не работает

- выполните `openchamber tunnel status --all`
- перезапустите туннель с того же инстанса/порта
- заново сгенерируйте ссылку подключения, если предыдущий токен уже использовался

См. [Туннели](/tunnels/) для полной настройки и [Обратный прокси](/reverse-proxy/), если вы его используете.

## Расширение VS Code не подключается

- подтвердите, что сервер OpenChamber запущен
- проверьте, что расширение обновлено
- перезагрузите окно VS Code и повторите подключение
