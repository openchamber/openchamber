# PWA и мобильный доступ

Веб-приложение OpenChamber устанавливается как приложение на телефон (PWA), чтобы вы могли держать его на главном экране и использовать на весь экран. Добавьте к нему [туннель](/tunnels/), и вы сможете проверять сессию откуда угодно.

## Установка

OpenChamber использует встроенную установку браузера, так что отдельной загрузки нет:

- **десктопный браузер** — используйте опцию **Install** в адресной строке
- **iPhone/iPad (Safari)** — «Поделиться» → **Add to Home Screen**
- **Android (Chrome)** — меню → **Install app** / **Add to Home Screen**

После установки оно открывается в собственном окне без браузерного chrome.

## Доступ с телефона

Чтобы открыть OpenChamber на телефоне, когда сервер работает на вашем компьютере, запустите [туннель](/tunnels/) и откройте ссылку (или отсканируйте QR-код) на телефоне. Используйте надёжный [пароль UI](/security/), когда делаете это.

## Мобильные настройки

В **Настройки → OpenChamber** есть несколько опций для мобильного и установленного режима — установленное имя приложения, ориентация экрана и поведение экранной клавиатуры.
