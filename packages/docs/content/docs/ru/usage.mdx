# Использование и квоты

Страница использования показывает, какую часть тарифа каждого провайдера вы уже использовали, чтобы видеть, насколько близко вы к лимиту. Открывается в **Settings → Usage**.

## Что вы увидите

Для каждого подключённого провайдера OpenChamber показывает:

- сколько из текущего окна вы уже использовали — в виде полосы
- разбивку по моделям
- индикатор темпа, чтобы понять, не подойдёте ли вы к лимиту раньше времени

Можно выбрать отображаемых провайдеров, а такая же сводка доступна из выпадающего меню в шапке приложения.

## Поддерживаемые провайдеры

Использование работает для провайдеров, публикующих квоту: Claude, Codex, GitHub Copilot, Google, OpenRouter, Kimi, NanoGPT, z.ai, Zhipu, MiniMax, Ollama Cloud и Wafer.

Использование по провайдеру появляется только после входа в него на странице [Провайдеры](/providers/). Некоторым провайдерам нужен дополнительный шаг — например, Ollama Cloud читает файл сессии, который вы настраиваете отдельно. Если по провайдеру нет данных, обычно это означает, что не хватает этих дополнительных учётных данных.
