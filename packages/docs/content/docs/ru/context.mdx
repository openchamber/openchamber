# Контекст

Каждая модель может удержать лишь ограниченный объём разговора за раз — её контекст. OpenChamber показывает, насколько он заполнен, чтобы вы понимали, когда сессия приближается к пределу и ответ может начать терять старые детали.

## Быстрый индикатор

Во время чата небольшой индикатор показывает процент использованного контекста. Он меняет цвет по мере заполнения:

- зелёный — места много
- жёлтый — заполняется (около трёх четвертей)
- красный — почти заполнен

Наведите курсор (или нажмите на мобильном), чтобы увидеть точное количество токенов.

## Полная панель контекста

Откройте вкладку **Context** в правой боковой панели для более полной картины текущей сессии:

- используемая модель и время начала сессии
- всего токенов относительно лимита модели
- итог по сообщениям и стоимости
- разбивка токенов последнего ответа
- примерное распределение того, что занимает контекст (ваши сообщения, сообщения агента, вывод инструментов)

Разбивка — это оценка, а не точный подсчёт — используйте её, чтобы понять, что заполняет окно, а не для биллинга.

## Что делать, когда контекст заполнен

Начните новую сессию для новой задачи, вместо того чтобы позволять одной сессии расти бесконечно. Короткий контекст быстрее и держит модель сфокусированной.
