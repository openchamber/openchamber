# Magic Prompts

OpenChamber использует встроенные промпты за кулисами всякий раз, когда делает что-то автоматически — пишет сообщение коммита, чертит PR, ревьюит issue, разрешает конфликт, суммаризует сессию. Magic Prompts — это место, где вы читаете и переписываете эти промпты. Откройте в **Настройки → Magic Prompts**.

Для обычного использования эта страница не нужна. Обращайтесь к ней, когда хотите, чтобы процесс вёл себя иначе — например, сообщения коммитов в определённом стиле.

## Редактирование промпта

1. Откройте **Настройки → Magic Prompts**.
2. Выберите промпт из групп в боковой панели — Git, GitHub, Planning и Session.
3. Отредактируйте текст и сохраните.

Некоторые промпты имеют видимую часть (сообщение, которое вы видите) и часть инструкций (скрытые указания для агента). В промптах могут быть `{{placeholders}}`, которые OpenChamber подставляет — например diff или заголовок issue — оставляйте их на месте.

## Сброс

Передумали? У каждого промпта есть **reset to default**, а также есть **reset all**, если хотите начать всё заново.
