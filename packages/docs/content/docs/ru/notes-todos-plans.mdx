# Заметки, задачи и планы проекта

У каждого проекта есть собственное пространство для заметок, список задач и сохранённые планы. Они принадлежат проекту, а не отдельной сессии, поэтому сохраняются при перемещении между сессиями. Найдите их на вкладке **Context** правой боковой панели (на мобильном — отдельная вкладка).

## Заметки

Свободное поле заметок для всего, что вы хотите запомнить о проекте. Сохраняется автоматически по мере ввода.

## Задачи

Простой чек-лист. Добавляйте пункты, отмечайте их, перетаскивайте для переупорядочивания и очищайте завершённые.

У каждой задачи есть меню **send**, чтобы передать её агенту:

- отправить в текущую сессию
- начать новую сессию с ней
- начать новую [worktree-сессию](/worktrees/) с ней (только когда проект — git-репозиторий)

## Планы

Место для хранения более длинных планов как сохранённых файлов. Вы можете:

- импортировать план из Markdown- или текстового файла
- открыть план для чтения в боковой панели
- удалить планы, которые больше не нужны

Вы должны вернуться на вкладку Context с сохранённой заметкой, отмеченной задачей или перечисленным планом — так вы поймёте, что действие применилось.
