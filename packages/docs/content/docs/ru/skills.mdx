# Навыки

Навык — это переиспользуемый набор инструкций, который агент может подключить при необходимости — например, «как мы пишем коммит-сообщения» или «наши API-конвенции». Управляются в **Settings → Skills**.

Чтобы установить готовые навыки вместо написания своих, см. [Каталог навыков](/skills-catalog/).

## Создать навык

1. Откройте **Settings → Skills**.
2. Создайте навык, дайте ему имя и короткое описание. По описанию агент решает, когда применять навык, поэтому будьте конкретны.
3. Напишите инструкции. Добавьте вспомогательные файлы, если они нужны.
4. Выберите, где он хранится:
   - **personal** — доступен в каждом проекте
   - **project** — доступен только в текущем проекте

## Использовать навык в чате

Введите `/` в середине сообщения, чтобы открыть выбор навыка, затем выберите нужный. Агент подгрузит инструкции этого навыка для ответа.

`/` в самом начале сообщения открывает [команды](/commands-snippets/).
