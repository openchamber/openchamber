# Документация OpenChamber

OpenChamber — это визуальная рабочая область вокруг OpenCode (AI-агента для кодинга, работающего в терминале). Она даёт экран, на котором можно наблюдать за работой и направлять её, вместо жизни в командной строке.

Используйте эти документы, чтобы:

- установить подходящее приложение под ваш способ работы
- открыть OpenChamber для безопасного удалённого использования
- настроить внешний вид и починить частые проблемы

## Для чего нужен OpenChamber

OpenChamber предназначен для тех частей AI-кодинга, которые выигрывают от «диспетчерской»: ветвящиеся сессии, ревью diff, управление терминалами, наблюдение за прогрессом инструментов, запуск действий проекта и удержание всей доски видимой, пока агент работает.
