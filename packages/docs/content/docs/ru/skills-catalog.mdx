# Каталог навыков

Каталог навыков позволяет устанавливать навыки, опубликованные другими, вместо написания собственных. Открывается в **Settings → Skills → Catalog**.

О написании собственных навыков см. [Навыки](/skills/).

## Установить навык

1. Откройте каталог.
2. Просмотрите встроенные источники — репозиторий Anthropic skills и сообщество ClawdHub — или воспользуйтесь поиском.
3. Выберите навык и установите его.
4. Выберите место установки: для всей вашей работы или только для текущего проекта.

Если навык с таким именем уже существует, OpenChamber спросит, что делать — пропустить, перезаписать или решать для каждого навыка отдельно.

## Добавить свой источник

Можно добавить любой Git-репозиторий как источник, используя имя `owner/repo` или полный Git URL. Для приватных репозиториев нужен доступ на вашей машине (SSH-ключ или сохранённые учётные данные); если источник не может аутентифицироваться, каталог сообщит об этом, а не промолчит.
