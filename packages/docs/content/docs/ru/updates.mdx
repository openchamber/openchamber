# Обновления

Способ обновления OpenChamber зависит от способа установки. В любом случае текущую версию можно посмотреть в **Settings → OpenChamber → About**.

## Desktop-приложение

Desktop-приложение проверяет обновления по релизам GitHub. Когда обновление доступно, OpenChamber сообщает вам, скачивает по вашему выбору и устанавливает при следующем перезапуске. Вы всегда контролируете процесс — ничего не устанавливается без вашего согласия.

## Веб / CLI

Если установлен CLI, обновите его в **About** кнопками проверки и обновления или из терминала:

```bash
openchamber update
```

OpenChamber определяет, как он был установлен (npm, pnpm, yarn или bun), и выполняет нужное обновление.

## Сервер OpenCode

OpenChamber и OpenCode обновляются отдельно. Когда доступна новая версия OpenCode, OpenChamber предлагает обновить её и затем перезапускает сервер. См. [OpenCode Server](/opencode-server/) о том, как этот сервер управляется.
