# Проекты

Проект — это папка на вашем компьютере, за которой следит OpenChamber, обычно одна кодовая база. Переключение проектов меняет папку, в которой работает агент, а также сессии и настройки этого проекта.

## Добавить проект

Проект можно добавить из нескольких мест:

- пункт **Add project** в палитре команд
- кнопка **+** в верхней части боковой панели сессий
- браузер папок при выборе каталога

Укажите папку, и OpenChamber запомнит её. Имя берётся из папки; позже его можно изменить.

## Переключить проекты

Выберите проект на боковой панели, чтобы сделать его активным. Всё — сессии, git, заметки — следует за открытым проектом.

## Сделать проект узнаваемым

Откройте **Settings → Projects**, чтобы задать проекту своё имя, цвет или значок. OpenChamber пытается найти значок автоматически — см. [Значки проектов](/project-icons/).

> В VS Code OpenChamber всегда использует открытую папку как единственный проект, поэтому добавлять или переключать нечего. Страница настроек Projects там скрыта.
