# ALIAS ADE Docs

ALIAS ADE is the visual workspace around OpenCode.

Use these docs to:

- install the right surface for your workflow
- expose ALIAS ADE safely for remote use
- customize appearance and troubleshoot common issues

## What ALIAS ADE is for

ALIAS ADE is for the parts of AI coding that benefit from a control room: branching sessions, reviewing diffs, managing terminals, watching tool progress, running project actions, and keeping the full board visible while the agent works.
